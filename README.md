# TrabajosPracticosProg2
Proyectos desarrollados con fines educativos para practicar el diseño de sistemas orientados a objetos en Java, aplicando principios de reutilización, extensibilidad y bajo acoplamiento. La implementación muestra cómo estructurar diversion sistemas cotidianos mediante abstracción, herencia y composición, facilitando la incorporación de nuevas funcionalidades
